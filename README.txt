1. Create VM with disk larger than 30G
2. Use UEFI for boot, rather than BIOS
3. Install git: "pacman -Sy git"
4. cd arch_installation directory and run ./install_dwm.sh
5. Follow the promt, the machine will reboot twice.
6. After last reboot change your resolution in .xinitrc
7. Finally start UI with startx.
